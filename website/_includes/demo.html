{%- comment -%} Embeds a live demo with Live / Source / Rendered HTML tabs. The Source tab hosts an inline Phlex / ViewComponent toggle so visitors can compare how the same UI is built in either engine. Both engine sources are rendered into the DOM; CSS hides the inactive one based on `data-engine` on the demo wrapper. The toggle script (in layout_end.html) syncs the choice across every demo on the page and persists it in localStorage.
